

# 💰 Budget Sava

### *Take Control of Your Money, Your Way*

---

## 📱 About the App

**Budget Sava** is a smart and user-friendly budgeting and financial tracking mobile application designed to help users manage their money more efficiently.

With powerful features like budget creation, expense tracking, savings insights, and premium upgrade options, Budget Sava gives users full control over their finances.

## 🛠 How It Works

1. **Landing Page**
   Welcomes users and provides options to sign up or log in.

2. **Sign-Up / Login**
   Users securely create accounts or log in to access their dashboard.

3. **Dashboard**
   Displays budgets, expenses, and savings in a clear and simple layout.

4. **Add Expense & Reports**
   Users can capture expenses and generate reports.

5. **Premium Plans Page**
   Offers subscription options with additional features like AI insights and an ad-free experience.

## ✨ Key Features

* ✔ Expense tracking by category
* ✔ Date range filtering for reports
* ✔ 📊 Graph visualization using MPAndroidChart
* ✔ 🎯 Min & Max budget goals display
* ✔ 🏆 Reward badge system for smart spending
* ✔ 📷 Expense photo attachment
* ✔ 💾 Offline storage using Room Database
* ✔ 📱 Optimized for real Android devices
